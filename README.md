# 📊 Brückner Servtec – Global Service Performance Dashboard

## Project Overview
This Power BI dashboard simulates a **global service performance tracking system** 
for an industrial machinery company. It was built using **synthetic data** generated 
to reflect realistic service operations.

## 🛠 Tools Used
- **Power BI** – Dashboard & visualizations
- **Python / Excel** – Synthetic data generation
- **CSV** – Data source files

## 📁 Files
| File | Description |
|------|-------------|
| `*.pbix` | Power BI dashboard file |
| `ServiceOrders.csv` | Synthetic service order data |
| `ServiceDetails.csv` | Detailed service records (revenue, margin, category) |

## 📌 Key Insights Tracked
- Revenue & Gross Margin by service category
- Units serviced by contract type (Framework, Warranty, EMI Finance)
- Service type breakdown (HMI Replacement, Drive Overhaul, Yield Improvement, etc.)
- Global service performance trends

## ⚠️ Note
All data in this project is **100% synthetic** and generated for portfolio purposes. 
It does not represent real Brückner Servtec data.
